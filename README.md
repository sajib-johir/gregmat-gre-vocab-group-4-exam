# GregMat GRE Vocab Group 1 Exam — Day 4

Offline GRE vocabulary exam portal for **Day 4: Words 91–120**.

## Word Range

Abstain, Approbation, Cherish, Corroborate, Disparate, Emulate, Enervate, Ephemeral, Fervid, Garrulous, Incendiary, Inimical, Intimate, Invigorate, Mitigate, Obsolete, Opaque, Paradigmatic, Pedantic, Placid, Polemical, Precipitate, Profundity, Prophetic, Prudent, Punctilious, Recondite, Scrupulous, Tranquil, Vacillate.

## Files

```text
index.html
style.css
script.js
README.md
```

## How to Run Locally

1. Extract the ZIP file.
2. Open `index.html` in a modern browser.
3. Enter your name.
4. Choose Basic, Medium, or Hard.
5. Read the instruction screen and start the exam.

No internet connection is required.

## Exam Levels

| Level | Name | Marks | Time |
|---|---|---:|---:|
| 1 | Basic Easy Recall Exam | 40 | 25 minutes |
| 2 | Medium Mixed Exam | 35 | 25 minutes |
| 3 | Hard GRE-Like Exam | 30 | 20 minutes |

## Features

- Offline HTML/CSS/JS portal
- Eye-soothing responsive UI
- Student name support
- Global countdown timer
- Auto-submit when time ends
- Part/page exam system
- Answers saved when moving between parts
- No hints or related vocabulary shown during the exam
- Selected options shown only by highlighting
- Clickable post-submission review
- Full question and all answer options shown after submission
- Student answer and correct answer shown
- Blank-wise review for double/triple blanks
- Sentence equivalence scoring
- Strong words and weak words section

## GitHub Pages Deployment

1. Create a GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and `README.md`.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose branch: `main`; folder: `/root`.
6. Save and wait for GitHub Pages to publish the site.

## Notes

The related vocabulary word is hidden during the exam and appears only after submission in the review cards.
